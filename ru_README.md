# Логин оператора

ОператорЛогин — это мод аутентификации NeoForge на стороне сервера для Minecraft 1.21.1. Он защищает настроенных игроков с помощью`/register <password>`и`/login <password>`прежде чем они смогут общаться, запускать другие команды, взаимодействовать с блоками или атаковать объекты.

## Текущая цель

- Загрузчик: NeoForge
- Майнкрафт: 1.21.1
- НеоФордж: 21.1.228
- Ява: 21

## Примечания по безопасности

- Пароли хранятся в`config/operatorlogin-passwords.properties`в виде соленых хешей PBKDF2-HMAC-SHA256.
— Устаревший простой текст и старые хеши SHA-256 переносятся в PBKDF2 после успешного входа в систему.
- Неаутентифицированные игроки могут использовать только`/login`и`/register`.
- Попытки входа в систему ограничены по скорости и имеют временную блокировку.
- Новые пароли должны соответствовать установленной минимальной длине и не могут быть пустыми или равными имени игрока.

## Конфигурация

Мод создает`config/operatorlogin.properties`при первом запуске.

| Option | Default | Description |
| --- | --- | --- |
| `authOnlyOperators` | `true` | If `true`, only operators must authenticate. Set to `false` to protect every player. |
| `kickTimeoutSeconds` | `60` | How long an unauthenticated player may stay connected. |
| `minPasswordLength` | `8` | Minimum accepted password length. |
| `maxLoginAttempts` | `5` | Failed attempts before lockout. |
| `lockoutSeconds` | `60` | Lockout duration after too many failed attempts. |

## Строить

```bash
./gradlew build
```

Мод jar производится в`build/libs/`.
